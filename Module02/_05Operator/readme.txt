Biểu thức là gì ?

Biếu thức là tập hợp các toán tử và toán hạng

- Toán hạng là các biến hoặc giá trị được thực hiện
- Toán tử là các phép toán cộng, trừ, nhân, chia, lấy phần dư, so sánh,...

Các loại Toán tử

- Toán tử gán: =, +=, -=, *=, /=, %=, .=
- Toán tử số học: +, -, \*, /, %, **, ++, --
- Toán tử so sánh: >, <, >=, <=, ==, ===, !=, !===
- Toán tử lý luận: &&, ||, ! (Độ ưu tiên NOT => AND => OR)
